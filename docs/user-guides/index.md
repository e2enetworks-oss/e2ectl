# User Guides

Use this section when you are operating `e2ectl` from a terminal, local script, or CI job.

The guides are split into two paths:

- task-first workflows for first-time and occasional users
- command-family guides for people who already know which resource they need

## Start Here

- [Quickstart](./quickstart.md) for the safest first path from install to a working profile
- [Create your first node](./workflows/first-node.md) for the first complete operator journey
- [Automation cookbook](./automation.md) if you are scripting from day one
